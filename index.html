<D3.js - Player Similarity

Bar Chart of player similarities

This page lets you choose a player from the drop down box (you can type in there as well, but mind the lag as it is loading thousands of names) to find similar players. It is my first try making something with D3, so it's far from perfect, but it works.

The files it uses are made by Ben Griffis. I just load the data and plot it, so all credits to him for the amazing work. You can find his documentation on Github


Click on the name in the y-axis to search for the player on transfermarkt.com. Click on the bar to go to the chart of similair players for that player.
